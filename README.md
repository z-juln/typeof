## 介绍

- 封装过的 typeof，比原生精明不少，且完美支持 typescript 类型提示
- 提供了 babel 插件，直接替换原生的 typeof

## 安装

```
$ npm install @juln/typeof
or
$ yarn add @juln/typeof
```
